Sitios Web Desarrollados

  • Aplicación Web de Turnos

    Desarrollé una página web estática diseñada para gestionar calendarios laborales basados en el sistema 3x4, el cual alterna entre semanas largas y semanas cortas. La aplicación permite al usuario visualizar de manera sencilla la rotación de horarios, identificar los días laborables y no laborables, y controlar sus turnos de trabajo con mayor organización. Este proyecto está construido con HTML, CSS y JavaScript, incorporando lógica para calcular automáticamente los ciclos semanales y ofrecer una experiencia clara, rápida y funcional.

  • Aplicación Web del Restaurante sobre ruedas

    Sobre Ruedas es un restaurante con un concepto único inspirado en la vibra y energía de los años 80, donde la comida no solo se disfruta, sino que se vive como una experiencia. Nuestro sello distintivo son nuestras meseras en patines, quienes llevan cada platillo directamente a tu mesa al mejor estilo retro, combinando sabor, diversión y nostalgia. El ambiente está cuidadosamente ambientado con colores neón, música ochentera y una estética vintage que transporta a nuestros clientes a una época llena de estilo y movimiento. En Sobre Ruedas ofrecemos un menú variado pensado para todos los gustos, un servicio rápido y dinámico, y una experiencia diferente que hace que cada visita sea inolvidable.

  • Aplicación Web del Restaurante sobre ruedas

    El proyecto consiste en el desarrollo de una clínica privada orientada a brindar servicios de salud de manera integral, eficiente y organizada, enfocándose en la optimización de la atención al paciente mediante el uso de tecnología y procesos administrativos estructurados. La clínica contará con 8 módulos administrativos interconectados que permitirán una gestión completa del funcionamiento interno, incluyendo administración general, expediente clínico digital, gestión de citas, facturación, punto de venta, control de pagos, inventario médico y generación de reportes. El sistema de gestión de citas está diseñado para mejorar la organización de la agenda médica, reducir tiempos de espera y garantizar una atención más eficiente. Asimismo, el módulo de punto de venta facilitará el registro y control de los servicios brindados, permitiendo transacciones claras, rápidas y seguras. El proyecto contempla instalaciones modernas, el uso de herramientas tecnológicas actualizadas y un enfoque centrado en la experiencia del paciente, asegurando un servicio de calidad, confiable y acorde a las necesidades de una clínica privada actual.
