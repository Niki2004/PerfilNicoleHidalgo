Sitios Web Desarrollados

  • Aplicación Web de Turnos

    Desarrollé una página web estática diseñada para gestionar calendarios laborales basados en el sistema 3x4, el cual alterna entre semanas largas y semanas cortas. La aplicación permite al usuario visualizar de manera sencilla la rotación de horarios, identificar los días laborables y no laborables, y controlar sus turnos de trabajo con mayor organización. Este proyecto está construido con HTML, CSS y JavaScript, incorporando lógica para calcular automáticamente los ciclos semanales y ofrecer una experiencia clara, rápida y funcional.
